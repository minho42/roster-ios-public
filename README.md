## How it works

### HealthRoster

1. Open a web browser.
2. Go to the HealthRoster, aka. Employee Online (EOL) website.
3. Log in to HealthRoster (required each time).
4. View the roster.

### Rosterr

1. Open an iOS app.
2. [Log in once.]
3. View the roster with above mentioned feature.
